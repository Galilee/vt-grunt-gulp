# Example d'utilisation avec Grunt / Gulp

## Installation des outils de CLI pour Grunt / Gulp
- `npm install gulp-cli -g`
- `npm install grunt-cli -g`

## Download des dependencies
- `npm install`

## Exécutions des tâches avec metrics
- `time gulp`
- `time grunt`
